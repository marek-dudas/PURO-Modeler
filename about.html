About this app

PURO Ontological Background Models

The main function of this app is graphical creation of PURO OBM.

OBM2OWL Transformation

The app can automatically create OWL ontology fragments that can describe the situation modeled in the OBM. Each ontology fragment will be created in different modeling style and visualized with WebVOWL. This allows you to easily see different OWL modeling options in one place.

Ontology local coverage comparison

Once created, an OBM can serve as a way of ontology local coverage visualization. Learn more.
